# Commission Legalization Reminder Automation
Project Overview

This project automates the daily monitoring of employee commission legalization deadlines.

Using Python, the system checks a commission database and automatically sends reminder emails to employees whose legalization due date matches the current date.

The automation also sends a notification email when no pending legalizations are detected.

Main Features
- Daily automated execution
- Date validation process
- Automatic reminder emails
- Notification system for no pending cases
- Process automation to reduce manual follow-up
  
Technologies Used
- Python
- Excel
- Pandas
- SMTP Email Automation
- Task Scheduler
  
Workflow
1. Read commission database
2. Compare legalization due dates with current date
3. Identify pending legalizations
4. Send automatic reminder emails
5. Generate notification when no emails are required



All data included in this repository has been anonymized and modified for portfolio purposes only.
